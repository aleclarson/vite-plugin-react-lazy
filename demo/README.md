## Instructions

1. Install dependencies
  ```
  pnpm i
  ```

2. Start the dev server
  ```
  pnpm run dev
  ```

3. Test the demo
  ```
  open http://localhost:3000
  ```

### Testing production bundle

1. Build the project
  ```
  pnpm run build
  ```

2. Test the demo
  ```
  pnpm run test
  ```
